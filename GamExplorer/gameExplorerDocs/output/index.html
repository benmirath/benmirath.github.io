Button HTML

Button HTML might vary, but to get the correct appearance, it must contain the button class:

		<button class="button">Button</button>

Also, links can be made to look like buttons. These are normal links, which also contain the button class:

		<p>Here is a link: <a href="http://www.  netmagazine.com" class="button">Link</a></p>

Button CSS

The CSS for buttons and button links is very straightforward - since it is simply hooked to the button class, the same CSS can be used for both buttons and links.
